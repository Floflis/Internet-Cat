Search the Web

Internet Cat isn't partnered with Microsoft/Bing. This is just the only search engine which works on iframes, the technology initially used by IC to load sites.

Soon, Internet Cat will support a variety of search engines. Not only on this start page and in address bar, you will be able to search even from your system-tray and desktop! In mobile browser, it will also have search widgets for launcher.

Internet Cat isn't yet ready, because it doesn't:

  • supports webview and other web advanced features
  • stores logins
  • saves sessions
  • records history
  • haves multi-tab browsing
  • support extensions

With a few of work, all of that will be implemented. And it will be the minimum, yet no more than our duty; soon there will be amazing features for your please ❤
